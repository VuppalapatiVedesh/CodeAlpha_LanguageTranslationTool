# 🌍 Language Translation Tool

This project is a **Language Translation Web Application** developed as part of the **CodeAlpha Artificial Intelligence Internship**.

The application allows users to enter text and translate it into multiple languages using **Python, Flask, and Deep Translator**.

---

## 🚀 Features

- 🌐 Translate text into multiple languages
- 🔍 Auto language detection
- 🧠 AI-powered translation using Deep Translator
- 📋 Copy translated text to clipboard
- 🧹 Clear text option
- 🎨 Clean and simple user interface

---

## 🛠 Technologies Used

- Python
- Flask
- Deep Translator
- HTML
- CSS

## 📸 Project Demo

The user can:

1. Enter text in the input box  
2. Select source language  
3. Select target language  
4. Click **Translate**  
5. View the translated result instantly
6. Copy the translated text
